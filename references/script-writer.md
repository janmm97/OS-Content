# How I Built: script-writer.md

## What This Document Is

A comprehensive scriptwriting reference that teaches the pipeline how to write short-form video ad scripts. It covers the current ad landscape, platform-specific requirements, hook types, storytelling structures, CTAs, and ad-type-specific rules (UGC vs Commercial Ad).

The pipeline reads this doc in Stages 1, 1b, and 2b — any time it's writing or revising a transcript.

---

## Why It Matters

This is the highest-leverage reference document. A weak script-writer.md produces scripts that are generic, badly paced, or off-brand. A strong one gives Claude the framework to write scripts that actually stop the scroll and convert.

## Step 1: Research with Perplexity

Run these queries and save the results:

**Platform and landscape data:**
```
What are the key stats and trends for short-form video advertising in 2025-2026? Include data on TikTok, Instagram Reels, and YouTube Shorts — completion rates, attention span data, CTR benchmarks, and what's changed vs 2023-2024.
```

**Hook writing:**
```
What are the most effective hook types for short-form video ads in 2026? Include specific hook templates, data on why they work, and examples for B2B SaaS or software tools. Cover: problem hooks, curiosity hooks, list hooks, result hooks, and contrarian hooks.
```

**Storytelling structures for ads:**
```
What storytelling structures work best for short-form video advertising? Cover PAS (Problem-Agitate-Solve), AIDA, BAB (Before-After-Bridge), and any 2025-2026 formats that are performing well. Include when to use each.
```

**CTAs for different funnel stages:**
```
What CTAs work best in short-form video ads across different funnel stages (awareness, consideration, conversion)? What's the difference in language, urgency, and placement between a TOFU and BOFU CTA?
```

**UGC vs produced ads:**
```
What's the difference between writing a UGC-style script and a traditional commercial ad script in 2026? Cover: tone, length, pacing, hook style, persona, and CTA approach. Include data on performance differences.
```

**ElevenLabs word count targets:**
```
What are the optimal word counts and sentence lengths for AI text-to-speech voiceovers at different video durations (15s, 30s, 60s)? How does pacing affect engagement in short-form video?
```

```
I'm building a scriptwriting reference for a content pipeline that generates short-form video ad scripts. The pipeline will read this doc to write scripts in two formats: UGC (person-to-camera, conversational) and Commercial Ad (voiceover hook, cinematic).

Write a comprehensive, instructional scriptwriting guide with these sections:

1. The 2026 Advertising Landscape — key stats, what's changed, why short-form still works
2. Platform Guide — TikTok, Reels, YouTube Shorts: format, duration, audience behavior, what performs
3. Core Scriptwriting Craft:
   - Hooks: the first 1–3 seconds (types, templates, why each works)
   - Emotional triggers and psychological principles
   - Storytelling structures with when-to-use guidance
   - Pacing and sentence rhythm
   - CTAs by funnel stage
4. UGC Script Rules — specific guidance for first-person, conversational, person-to-camera scripts
5. Commercial Ad Script Rules — voiceover hook-only format, cinematic, emotional
6. Word count and pacing targets for AI TTS (15s, 30s formats)
7. Common mistakes to avoid

Write it as instructional guidance an AI will follow directly. Be specific — templates, examples, and rules, not descriptions.
```

## Step 4: Refine Against Pipeline Output

Run 10+ rows through the pipeline and evaluate each transcript on:
- **Hook quality** — does it stop the scroll in the first 3 seconds?
- **Pacing** — does it flow naturally when read aloud?
- **Brand alignment** — does it sound like your brand?
- **CTA** — is it clear and action-driving?

For each miss, trace it back to the script-writer.md section that should have prevented it, and add a rule or example.

---

## What a Good Version Includes

- Platform-specific word count and pacing targets (15s = ~35 words, 30s = ~70 words)
- At least 6 named hook types with templates and examples
- At least 3 storytelling frameworks with when-to-use guidance
- Explicit UGC vs Commercial Ad rules — they should be clearly separate sections
- CTAs by funnel stage with example language
- A "read it aloud" quality check rule
- Your brand's persona and voice rules in a dedicated section
